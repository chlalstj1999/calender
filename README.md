# 일정 관리 프로젝트

이 프로젝트는 JSP(JavaServer Pages)를 사용하여 개발한 간단한 일정 관리 애플리케이션입니다. 사용자는 일정을 추가, 수정, 삭제하고, 전체 일정을 조회할 수 있습니다.

## 목차

- [기능](#기능)
- [기술 스택](#기술-스택)

## 기능

- 일정 추가
- 일정 수정
- 일정 삭제
- 일정 조회 ( 팀장급은 팀원들의 일정도 같이 조회 )
- 사용자 친화적인 UI

## 기술 스택

- **프로그래밍 언어**: javascript
- **웹 프레임워크**: JSP
- **데이터베이스**: mariaDB
- **기타**: HTML, CSS, JavaScript
